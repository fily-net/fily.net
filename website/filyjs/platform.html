  • Zeanium
    Zeanium是一套完整的javascript面向对象(OOP)的底层开发框架，可以很自由的定义一个包含完整特性如静态类(static)、 静态属性(statics)、 属性(property)、事件(event)、方法(method)的JavaScript对象， zeanium核心功能主要有动态定义Class(面向对象)、Promise(异步调用)实现、Require(模块加载机制)、Plugin(插件机制)、Data Model(数据模型)、TestSuit(针对Zeanium的测试套件)。

  • Zeanium-Node
    基于Zeanium核心框架实现的服务器后端Server框架，它提供了基本的HttpServer、Socket、WebSocket的实现、Session(会话机制)、Filter(过滤机制)、Cache(缓存机制)、ORM(数据库关系对象映射)， 开发者只需要通过使用Zeanium-Node框架可以只关注于具体业务的情况下，很轻松的开发属于自己的Application Backend Service。 Node-Server实现Http Container机制, 可以在一个服务下部署多个应用，同时支持应用程序热部署。对于高级开发者，框架底层提供功能强大的插件机制，可以根据自己的业务编写业务需要的HttpRequestHandler(请求处理器)。
    Git: git
    Doc: doc
    Issues: Issues
  • Zeanium-UI
    基于Zeanium核心框架实现的浏览器前端UI框架, 通过底层UI框架可以自定义组件和前端业务模块，框架提供常用UI组件库和一套zeanium定义的皮肤样式。
